# Hapteec UI

## 開発サーバー

開発サーバーを実行するには、`npm run start` を実行します。 `http://localhost:4200/` にアクセスしてください。
ソースファイルを変更すると、アプリケーションが自動的にリロードされます。

## ビルド

プロジェクトをビルドするには、`ng build` を実行します。ビルドアーティファクトは `dist/` ディレクトリに保存されます。
